**2. Create PlayerTimerBar** (`apps/web/app/join/[roomCode]/game/PlayerTimerBar.tsx`):
- `'use client'`
- Props: `{ duration: number; startedAt: number; active: boolean }`
- Layout: `fixed top-0 inset-x-0 h-[6px] bg-gray-200 z-50` (thin bar at very top per D-05)
- Inner fill: `<div>` with:
  - `className="h-full bg-indigo-500"`
  - `style={{ transform: \`scaleX(${progress})\`, transformOrigin: 'right', transition: 'transform 100ms linear' }}`
  - `transformOrigin: 'right'` — in RTL context, the timer depletes from the right side (inline-start in Arabic), which is the natural reading direction per RTL-05
- Use `useState` + `useEffect` with `setInterval(100)` to update progress:
  ```typescript
  const [progress, setProgress] = useState(1)
  useEffect(() => {
    if (!active) return
    const interval = setInterval(() => {
      const elapsed = (Date.now() - startedAt) / 1000
      const remaining = Math.max(0, 1 - elapsed / duration)
      setProgress(remaining)
    }, 100)
    return () => clearInterval(interval)
  }, [duration, startedAt, active])
  ```
- When `active === false`: freeze at current progress
- Accessibility: `role="progressbar" aria-valuenow={Math.ceil(progress * duration)} aria-valuemax={duration} aria-label="الوقت المتبقي"`

**2. Extend socket listeners** (in the existing `useEffect` that runs when `phase !== 'form'`):
```typescript
socket.on('question:start', ({ question, questionIndex: qi, total, hostSettings: hs }) => {
  setCurrentQuestion(question)
  setQuestionIndex(qi)
  setHostSettings(hs)
  setMyAnswer(null)
  setCorrectIndex(null)
  setPlayerPhase('answering')
  setQuestionStartedAt(Date.now())
  setPhase('playing')
})

socket.on('question:revealed', ({ correctAnswerIndex, playerResults }) => {
  setCorrectIndex(correctAnswerIndex)
  setPlayerPhase('revealed')
  // Find my score from playerResults using myToken
  const myResult = playerResults.find((p: any) => p.id === myToken)
  if (myResult) {
    setMyScore(myResult.score)
    setMyStreak(myResult.streak)
  }
})

socket.on('game:podium', ({ top3 }) => {
  setPhase('ended')
})
```

Add cleanup in return: `socket.off('question:start'); socket.off('question:revealed'); socket.off('game:podium')`
