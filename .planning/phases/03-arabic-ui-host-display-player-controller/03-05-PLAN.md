**1. Create PodiumScreen** (`apps/web/app/host/[roomCode]/game/PodiumScreen.tsx`):
- `'use client'`
- Import `* as m from 'motion/react-m'`
- Props:
  ```typescript
  interface PodiumScreenProps {
    top3: Array<{ id: string; name: string; emoji: string; score: number; rank: number }>
  }
  ```
- Full-screen dark layout: `w-full h-full flex flex-col items-center justify-center bg-gray-950`
- Heading: `<h1 className="text-4xl font-black text-white mb-12">المتصدرون</h1>`
- Podium container with staggered animation:
  ```tsx
  const containerVariants = {
    hidden: {},
    show: {
      transition: { staggerChildren: 0.4, staggerDirection: -1 }
    }
  }

  const itemVariants = {
    hidden: { y: 40, opacity: 0 },
    show: { y: 0, opacity: 1, transition: { duration: 0.5, ease: 'easeOut' } }
  }
  ```
- Render top 3 in ORDER [3rd, 2nd, 1st] (staggerDirection: -1 means 1st appears last for drama):
  - Actually with staggerDirection: -1, the LAST child animates first. So render in order [1st, 2nd, 3rd] and staggerDirection: -1 will show 3rd first, then 2nd, then 1st.
  - Each podium entry:
    ```tsx
    <m.div variants={itemVariants} className="flex flex-col items-center gap-3">
      <span className="text-6xl">{entry.emoji}</span>
      <span className="text-2xl font-bold text-white text-start">{entry.name}</span>
      <span className="text-xl font-semibold text-indigo-300">{entry.score} نقطة</span>
      {entry.rank === 1 && <span className="text-4xl">🥇</span>}
      {entry.rank === 2 && <span className="text-3xl">🥈</span>}
      {entry.rank === 3 && <span className="text-2xl">🥉</span>}
    </m.div>
    ```
- Visual podium bars: Create a row of 3 columns (flex row), each column represents the "podium height":
  - 1st place: tallest bar (h-40)
  - 2nd place: medium bar (h-32)
  - 3rd place: shortest bar (h-24)
  - Arranged visually as: [2nd] [1st] [3rd] (classic podium layout)
  - Bar color: `bg-gray-800 rounded-t-2xl`
  - Player info sits on top of their bar
- `<m.div variants={containerVariants} initial="hidden" animate="show" className="flex items-end justify-center gap-6 mt-8">`
- Handle edge cases: if `top3.length < 3`, only show available players (per UI-SPEC empty states)
- `prefers-reduced-motion`: replace y/opacity animation with instant display (opacity only, no y)
