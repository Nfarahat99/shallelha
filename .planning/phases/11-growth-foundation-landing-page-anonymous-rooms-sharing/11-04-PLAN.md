   - Room code: "كود الغرفة: {roomCode}" in white/70%
   - "اضغط للانضمام" call to action
   - Use inline styles (satori requires inline styles, NOT Tailwind)

6. Load Cairo font from Google Fonts CDN (same approach as existing server cards route):
   ```
   const cairoFont = await fetch(
     'https://fonts.gstatic.com/s/cairo/v31/SLXVc1nY6HkvangtZmpcWmhzfH5lWWgcQyyS4J0.ttf'
   ).then(r => r.arrayBuffer())
   ```

7. Return:
   ```
   return new ImageResponse(jsx, {
     width, height,
     fonts: [{ name: 'Cairo', data: cairoFont, style: 'normal', weight: 700 }],
   })
   ```
