**3. Add a useEffect that runs when the session loads and the name input is still empty:**
```typescript
useEffect(() => {
  if (session?.user) {
    // Pre-fill name from profile displayName, fall back to OAuth name
    const profileName = (session.user as { displayName?: string }).displayName ?? session.user.name ?? ''
    if (profileName && !name) {
      setName(profileName)
    }
    // Pre-fill emoji if user has a saved avatarEmoji
    const profileEmoji = (session.user as { avatarEmoji?: string }).avatarEmoji
    if (profileEmoji && !selectedEmoji) {
      setSelectedEmoji(profileEmoji)
    }
  }
}, [session])
```

**Note on type casting:** NextAuth's default session.user type only has `name`, `email`, `image`.
