# Plan 08-01: Error Boundaries, Loading States & Process Resilience

## Goal
All routes have loading skeletons and error boundaries; UI components show structured Arabic error panels, disconnect banners, and loading spinners; server handles process-level failures gracefully.

## Prerequisites
None — this plan can start immediately. No dependency on other Phase 8 plans.

## Threat Model
<threat_model>
| Threat | STRIDE | Mitigation |
|--------|--------|------------|
| Unhandled server exception causes silent crash | Denial of Service | `process.on('uncaughtException')` logs `[FATAL]` and exits; Railway ON_FAILURE restarts |
| Timer memory leak from abandoned games | Denial of Service | Disconnect handler clears `autoRevealTimers` and `votingTimers` for host's room |
| Error digest/stack leak to client | Information Disclosure | `error.tsx` shows only `error.message`, never `error.stack`; digest is safe (Next.js internal) |
</threat_model>

### Task 3: SkeletonCard, LoadingButton, ErrorBanner, DisconnectBanner components
**File(s):** `apps/web/components/ui/SkeletonCard.tsx`, `apps/web/components/ui/LoadingButton.tsx`, `apps/web/components/ui/ErrorBanner.tsx`, `apps/web/components/ui/DisconnectBanner.tsx`
**What:**
- **SkeletonCard**: `'use client'`. Props: `{ count?: number }` (default 3). Renders `count` skeleton rows, each `w-full h-[56px] bg-gray-200 rounded-xl animate-pulse`. Rows 2+ get decreasing opacity (0.7, 0.4). Container: `space-y-2 pointer-events-none`.
- **LoadingButton**: `'use client'`. Props: `{ loading: boolean, children: React.ReactNode, className?: string } & ButtonHTMLAttributes<HTMLButtonElement>`. When `loading=true`: adds `opacity-70 pointer-events-none cursor-not-allowed`, replaces children with inline spinner (`h-5 w-5 border-2 border-white border-t-transparent rounded-full animate-spin inline-block`) + `gap-2` + "جارٍ الانضمام...". Spinner has `aria-hidden="true"`. Button has `min-h-[44px]`.
- **ErrorBanner**: `'use client'`. Props: `{ type: 'full-room' | 'invalid-code', onDismiss?: () => void }`. Renders `w-full rounded-xl bg-red-50 border border-red-200 px-4 py-3 flex items-start gap-3`. SVG warning/X-circle icon (20px, `text-red-500 shrink-0`). Headline + body per UI-SPEC section 7.2. Has `role="alert"`.
- **DisconnectBanner**: `'use client'`. Manages socket `disconnect`/`connect` events via `useEffect`. States: hidden, disconnected (red `bg-red-600`), reconnected (green `bg-green-600`). Fixed position `top-[6px] inset-x-0 z-40`. Shows spinner when disconnected, check icon when reconnected. Arabic copy per UI-SPEC 7.2. Has `role="alert"`. Success state lasts 1500ms then fades out with `opacity-0 transition-opacity duration-300`. Remove from DOM after 1800ms total.
**Acceptance:** `cd apps/web && npx next build` succeeds. All 4 component files exist and export named components.

### Task 5: Reduced motion CSS + process resilience
**File(s):** `apps/web/app/globals.css`, `apps/server/src/index.ts`, `apps/server/src/socket/index.ts`
**What:**
- **globals.css**: Append `@media (prefers-reduced-motion: reduce)` block. Inside: `.animate-pulse { animation: none; opacity: 0.7; }` and `.animate-spin { animation: none; }`.
- **index.ts** (server): Add before `httpServer.listen()`:
  ```
  process.on('uncaughtException', (err) => { console.error('[FATAL] Uncaught exception:', err); process.exit(1); })
  process.on('unhandledRejection', (reason) => { console.error('[ERROR] Unhandled promise rejection:', reason); })
  ```
- **socket/index.ts**: In the `disconnect` handler, after the existing log line: check `socket.data.isHost && socket.data.roomCode`. If true, call `clearAutoRevealTimer(socket.data.roomCode)` and `clearVotingTimer(socket.data.roomCode)`. Import these functions from `./game` (they must be exported — add `export` keyword to both in `game.ts` if not already exported).
**Acceptance:** `cd apps/server && npm run test` passes. `cd apps/web && npx next build` succeeds. `grep -c "uncaughtException" apps/server/src/index.ts` returns 1. `grep -c "prefers-reduced-motion" apps/web/app/globals.css` returns 1.
