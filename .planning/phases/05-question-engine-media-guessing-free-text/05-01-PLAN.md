    1. **game.service.ts** — Add `calculateFreeTextScore` function (per D-09):
       ```typescript
       import type { FreeTextAnswer } from './game.types'
       
       export interface FreeTextResult {
         authorScores: Record<string, number>
         voterScores: Record<string, number>
         winnerIds: string[]
         winnerText: string
       }
       
       export function calculateFreeTextScore(
         freeTextAnswers: Record<string, FreeTextAnswer>,
       ): FreeTextResult
       ```
       Logic:
       - Find maxVotes across all answers
       - If maxVotes === 0: return empty winnerIds, all scores 0, winnerText empty
       - All answers with votes.length === maxVotes are winners
       - Winner authors get 800 points each
       - Non-winner authors get 0
       - Voters who voted for a winning answer get 200 bonus
       - Voters who voted for a losing answer get 0 bonus
       - winnerText = text of first winner (or last iterated — not critical)
       - Ties: ALL tied authors get 800, ALL voters for ANY tied winner get 200
