    1. Start the server: `cd apps/server && npm run dev`
    2. Start the web app: `cd apps/web && npm run dev`
    3. Open host screen at `http://localhost:3000/host` in a desktop browser
    4. Open player screen at `http://localhost:3000/join/{roomCode}` in a mobile browser (or mobile emulator)
    5. Join with 2+ players

    **Test 1: Media Guessing — Image Blur**
    1. Start a game (ensure seed includes MEDIA_GUESSING questions)
    2. When a media guessing question appears:
       - Host screen: verify image starts fully blurred (CSS blur 20px)
       - Host screen: verify image progressively unblurs over the timer duration (smooth CSS transition)
       - Player phone: verify same blurred image appears
       - Player phone: verify 4 answer options appear below the image (same layout as MC)
    3. Answer as a player — verify speed-based scoring works (same as MC)
    4. Host clicks reveal — verify image instantly unblurs (fast snap)

    **Test 2: Media Guessing — Audio (if audio seed URLs exist)**
    1. When an audio question appears:
       - Verify audio auto-plays (or fallback play button appears)
       - Verify badge with pulsing animation on host screen
       - Verify 4 answer options appear (same MC layout)

    **Test 3: Free Text Entry**
    1. When a free text question appears:
       - Player phone: verify textarea with Arabic placeholder
       - Player phone: verify submit button is disabled (no text entered)
       - Type an answer — verify button enables
       - Submit — verify waiting screen appears with submitted text
    2. Host screen: verify answers appear in live feed (emoji + text)
    3. Timer expires or host clicks lock button:
       - Verify voting phase starts on both screens
       - Player phone: verify VotingUI shows all answers
       - Verify player cannot vote for their own answer (greyed out)
       - Vote for another answer — verify selection highlights
    4. After 15s voting timer:
       - Verify winning answer highlighted on host screen
       - Verify scores updated (author +800, voters for winner +200)

    **Test 4: Mixed Game Flow**
    1. Play through a game with MC + media guessing + free text questions
    2. Verify game transitions smoothly between question types
    3. Verify leaderboard shows after each question (all types)
    4. Verify game ends normally with podium

    1. All new text is Arabic, right-aligned
    2. Free text textarea has correct RTL direction
    3. No layout breaks in any new component
